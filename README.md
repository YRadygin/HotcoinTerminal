# Hotcoin Terminal (каркас, v0.1)

Десктопный торговый терминал под биржу Hotcoin. **WinUI 3 / .NET 8**, тёмная тема,
Mica-фон окна + акриловые «стеклянные» панели, скруглённые контролы.

Текущий этап — **каркас с графическими заглушками**: все данные (свечи, сигналы
скринера, события, баланс) генерируются локально в `Services/StubDataService.cs`.
Сервисный слой Hotcoin API и интеграция Claude API будут добавлены следующими этапами.

## Требования для сборки

- Windows 10 1809+ или Windows 11 (Mica-эффект полноценно работает на Windows 11)
- Visual Studio 2022 (17.8+) с компонентами:
  - рабочая нагрузка **«Разработка классических приложений .NET»**
  - в ней отмечен пункт **Windows App SDK C# Templates / средства WinUI**
- .NET 8 SDK

## Запуск

1. Открыть `HotcoinTerminal.sln` в Visual Studio.
2. Выбрать платформу **x64** в панели конфигурации.
3. F5. Проект настроен как **unpackaged** (`WindowsPackageType=None`),
   поэтому запускается как обычное приложение, без MSIX-упаковки.

## Что уже работает

- Две вкладки в титлбаре: **Аналитика** (полный макет) и **Торговля** (плейсхолдер).
- Скринер: чипы-фильтры по стратегиям (Grid / Mean Rev / Momentum / События),
  поиск по паре, таблица сигналов со скоринг-бейджами.
- Выбор пары в скринере обновляет заголовок и график.
- Свечной график на Canvas (без внешних библиотек): свечи, MA(10), объём,
  RSI(14) с уровнями 30/70. Переключение таймфреймов перегенерирует стаб-данные.
- Карточка **AI-анализ** (будущий вывод Claude API) с кнопкой
  **«Углублённый AI-анализ»** — открывает отдельное окно-заглушку, где позже
  будут подробные рекомендации, уровни входа/стопа/целей и тайминги удержания.
- Лента событий, детали пары, сводка баланса, статус-бар.

## Структура

```
HotcoinTerminal/
  App.xaml(.cs)              — тема, палитра, общие стили (стекло, чипы, кнопки)
  MainWindow.xaml(.cs)       — Mica, кастомный титлбар, вкладки, статус-бар
  Views/
    AnalyticsPage.xaml(.cs)  — аналитическая вкладка (скринер, график, AI-карточка)
    TradingPage.xaml(.cs)    — торговая вкладка (плейсхолдер)
    DeepAnalysisWindow.xaml(.cs) — окно углублённого AI-анализа (заглушка)
  Controls/
    CandleChartControl.xaml(.cs) — свечной график на Canvas
  Models/Models.cs           — Candle, SignalRow, EventItem
  Services/StubDataService.cs — временный генератор данных
  Helpers/Palette.cs         — палитра цветов
```

## Дорожная карта (следующие этапы)

1. Сервисный слой Hotcoin API: REST (тикеры, свечи, баланс) + WebSocket (живые цены).
2. Движок стратегий (`IStrategy`) и реальный цикл сканирования.
3. Интеграция Claude API: анализ сетапов, окно углублённого анализа.
4. Торговая вкладка: сделки, P&L, оценка активов.
5. Опционально: замена самописного графика на ScottPlot.WinUI или
   TradingView Lightweight Charts через WebView2.

## Дисклеймер

Проект предназначен для аналитики и обучения. Торговля криптовалютой связана
с высоким риском; никакая стратегия или AI-анализ не гарантируют прибыль.
